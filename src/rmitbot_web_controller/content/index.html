System Instructions

Manual Mode:

  • Keyboard: Use WASD keys to drive, QE to rotate, Space to stop
  • D-Pad: Click and hold directional buttons for precise control
  • Speed Control: Adjust slider between 0-1.0 m/s
  • Building Maps: Drive around to create a map with SLAM. Save it from the map selector

Auto Mode:

  • Load Map: Click "Manage Maps" to select and load a saved map
  • Set Goals: Enter coordinates (X, Y, Yaw) or click on the map visualization
  • Navigation: Click GO to start. Watch progress bar and distance feedback
  • Cancel: Click "Cancel Navigation" anytime to stop

Map Management:

  • View Maps: Click "Manage Maps" to see all saved maps
  • Load Map: Select from list to load for navigation
  • Preview: View map before loading (coming soon)

System Features:

  • Camera Feed: Live video from robot's front camera
  • Map View: Real-time SLAM visualization showing robot position
  • Status Indicator: Green dot = Connected, Red dot = Offline
  • Mode Badge: Shows current operation mode (top-right header)
